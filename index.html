How to Use the Pallet Stack Optimizer

This tool helps you find the most efficient way to stack items on a pallet. Adjust the sliders to match your bag dimensions, weight, and pallet constraints, then click "Optimize Stack."

  • Bag Dimensions: Set the length, width, height, and weight of the individual bags.
  • Pallet Dimensions: Set the length, width, and maximum height of the pallet.
  • Max Gross Weight: Set the maximum allowable weight of the pallet including the bags.
